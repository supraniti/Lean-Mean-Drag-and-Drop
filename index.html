Lean & Mean Drag and Drop is a small script for dragging, dropping, sorting and reordering html structures

Features
Support for deep nested structures ('nestable sortables')
Smooth transitions
Auto scroll while dragging
Lightweight (~3.5kb gzipped)
No dependencies
Super easy to integrate with your own app
Supports touch events
Compatible with all modern browsers
Can work with any type of layout
Installation

Add mldd.min.js to your project and load it.

Initialize LMDD instance with your preferred options, and make sure to listen to the relevant events and respond accordingly

lmdd.set(document.getElementById('yourMarkupID'),{yourOptionsObject});
Options

lmdd.set(document.getElementById('someID'), {
    containerClass:'lmdd-container',//choose any class to act as a container class, a container can nest draggable items inside
    draggableItemClass: 'lmdd-draggable',//choose any class to act as a draggable item, a draggable can be moved inside and between containers
    handleClass: false,//if you want a handle, just specify its class
    dragstartTimeout: 50,//small delay on initial execution to let the user select text
    calcInterval: 200,//interval between position calculations(ms), small amount of time means faster reaction and more CPU
    revert: true,//when revert is set to true the draggable will maintain its original position when the cursor is out of container bounds
    nativeScroll: false,//lmdd uses a scroll control function, you can eliminate it and use the native browser scroll
    mirrorMinHeight: 100,//
    mirrorMaxWidth: 500,//set the maximum width and minimum height of the draggable mirror (lmdd will scale the mirror to this size for better UX while dragging large items)
    matchObject: false,//explain please... :-)
    positionDelay: false,//setting to 'true' will make dragging a bit less responsive but will prevent flickering when moving nested structures
    dataMode: false//setting to 'true' will undo all DOM manipulations on drag end, this is useful when integrating with react/angular/vue
});

          Example #1 - Simple Sortable Lists

          Note LMDD is markup oriented, meaning your HTML markup determines the way drag and drop behaviour:

          Elements marked with class="your-draggable-class" will be draggable

          Elements marked with class="your-container-class" will act as containers and you will be able to drop draggables inside them

          HTML markup:

          <div id="simple-sortable-1">
  <div class="container">
      <div class="orange rectangle"></div>
      <div class="light-green rectangle"></div>
      <div class="pink rectangle"></div>
      <div class="deep-purple rectangle"></div>
      <div class="light-blue rectangle"></div>
      <div class="lime rectangle"></div>
  </div>
</div>

          Javascript:

          lmdd.init(document.getElementById('yourMarkupID'),{yourOptionsObject});

          Example #2 - Containers, sortable, and draggable items

          LMDD is markup oriented, meaning you use the markup to tell the app what to act like...

          • html element marked with 'lmdd-container' class will act as a drop target
          • html element marked with 'lmdd-draggable' class will set the element as draggable so you can move it around and drop it in containers
          • html element marked with 'lmdd-sortable' class means this is a sortable item (you can drop the draggable before or after this element
